# Data Modeling II


## Getting Started

##### directory to project 02-data-modeling-ii
```sh
$ cd 02-data-modeling-ii
```

##### Started
```sh
python -m venv ENV
source ENV/bin/activate
pip install -r requirements.txt
```

## Running Cassandra

```sh
docker-compose up
```

To shutdown, press Ctrl+C and run:

```sh
docker-compose down
```

## Create table ,Insert and select data

python etl.py
